SPINNERS

Los spinners de Bootstrap 5 son componentes utilizados para indicar a los usuarios que un proceso está en curso. Estos son elementos visuales que proporcionan una retroalimentación visual de que algo está sucediendo, como la carga de datos o el procesamiento de una solicitud.

COLORED SPINNERS

Estos spinners (indicadores de carga) están disponibles en diferentes colores para adaptarse a la paleta de colores de Bootstrap y proporcionar una retroalimentación visual más agradable y coherente con el diseño de la aplicación.

GROWING SPINNERS

CARDS

Las tarjetas (cards) de Bootstrap 5 son componentes flexibles y extensibles que se utilizan para mostrar contenido en un contenedor con bordes y sombreados. Las tarjetas pueden contener diferentes elementos como texto, imágenes, listas de elementos, enlaces y más.

CONTEXTUAL CARDS

Las "Contextual Cards" en Bootstrap 5 son una variante de las tarjetas estándar que están diseñadas para resaltar información importante o proporcionar contexto adicional mediante el uso de colores y estilos específicos. Estas tarjetas se utilizan típicamente para resaltar contenido crítico, como alertas, mensajes de estado o información clave que requiere atención especial.

CARD IMAGES

Las tarjetas con imágenes (Cards images) en Bootstrap 5 son componentes que permiten mostrar imágenes de manera atractiva junto con contenido relacionado, como texto, títulos y enlaces. Estas tarjetas son muy útiles para presentar información visualmente atractiva, como productos, perfiles de usuario, publicaciones de blog, entre otros.

                    </div>
                  </div>

TOAST

Un toast de Bootstrap es un componente de notificación que se despliega en la parte superior o inferior de la pantalla para mostrar mensajes breves al usuario. Estos mensajes pueden ser informativos, de éxito, advertencias o errores. Los toasts de Bootstrap son altamente personalizables y pueden contener texto, enlaces o incluso botones para interactuar con el usuario. 


                  <div class="toast show">
                    <div class="toast-header">
                      Toast Header
                      <button type="button" class="btn-close" data-bs-dismiss="toast"></button>
                    </div>
                    <div class="toast-body">
                      Some text inside the toast body
                    </div>
                  </div>

OFFCANVAS

El componente "offcanvas" en Bootstrap 5 permite crear un área de contenido que se desliza en y fuera de la pantalla. Es útil para menús laterales, paneles de información, o cualquier tipo de contenido que necesite aparecer desde un lado de la ventana del navegador.


                <!-- Offcanvas Sidebar -->
                <div class="offcanvas offcanvas-start" id="demo">
                  <div class="offcanvas-header">
                    <h1 class="offcanvas-title">Heading</h1>
                    <button type="button" class="btn-close text-reset" data-bs-dismiss="offcanvas"></button>
                  </div>
                  <div class="offcanvas-body">
                    <p>Some text lorem ipsum.</p>
                    <p>Some text lorem ipsum.</p>
                    <button class="btn btn-secondary" type="button">A Button</button>
                  </div>
                </div>
              
                <!-- Button to open the offcanvas sidebar -->
                <button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" data-bs-target="#demo">
                  Open Offcanvas Sidebar
                </button>

La clase .offcanvas crea la barra lateral offcanvas. La clase .offcanvas-start posiciona el offcanvas y lo hace de 400px de ancho. Consulta los ejemplos a continuación para ver más clases de posicionamiento.
La clase .offcanvas-title asegura márgenes y altura de línea adecuados.
Luego, agrega tu contenido dentro de la clase .offcanvas-body.
Para abrir la barra lateral offcanvas, debes usar un elemento ' <button>' o un elemento '<a>' que apunte al id del contenedor .offcanvas (#demo en nuestro ejemplo).
Para abrir la barra lateral offcanvas con un elemento '<a>', puedes apuntar a #demo con el atributo href, en lugar del atributo data-bs-target.



PAGINATION

La paginación en Bootstrap 5 se utiliza para dividir contenido grande en páginas más manejables, permitiendo a los usuarios navegar fácilmente a través de ellas. Bootstrap 5 proporciona componentes y clases predefinidas para crear paginaciones estilizadas de forma rápida y sencilla.
